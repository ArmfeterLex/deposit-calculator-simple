# Deposit Calculator Simple

Простой калькулятор депозита на Python. Рассчитывает остаток на счете через 1, 2 и 3 года при фиксированной ставке 4% годовых.

## Функциональность

- Ввод первоначальной суммы депозита
- Расчет остатка на счете через 1 год
- Расчет остатка на счете через 2 года
- Расчет остатка на счете через 3 года
- Фиксированная процентная ставка 4% годовых
- Округление результатов до двух знаков

## Пример работы

Первоначальный депозит: 1000
1 год: 1040.0
2 год: 1081.6
3 год: 1124.86

## Код

def calc_s(init_deposit):
    int_rate = 0.04
    balance1 = round(init_deposit * (1 + int_rate), 2)
    balance2 = round(balance1 * (1 + int_rate), 2)
    balance3 = round(balance2 * (1 + int_rate), 2)
    print(f"1 год: {balance1}")
    print(f"2 год: {balance2}")
    print(f"3 год: {balance3}")

init_deposit = float(input("Первоначальный депозит: "))
calc_s(init_deposit)

## Примечание

Программа использует фиксированную ставку 4%. Для изменения ставки достаточно поменять значение переменной int_rate в коде.
